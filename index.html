Bakasura Website Framework.

The Bakasura framework is a web site creator that allows you to add pages and fragments to your site to incrementally build your content with simple and consistent formatting. Bakasura can also utilise plugins that can generate their own fragments or pages. This means it's extensible for developers to create their own page or fragment types (like a blog post fragment or comic book page). It is currently a work in progress and in alpha stage build.

Requirements

To start with the framework you need to have the following prerequisites:

  • PHP 5.6 and higher
  • Apache 2.2 and higher
  • MySQL database
  • That's it.
